# SEO 檢查清單

完整的 SEO 審核和優化檢查清單，確保您的網站符合所有 SEO 最佳實踐。

---

## 📋 頁面發布前檢查清單

### 基本 Meta 標籤
- [ ] 設定唯一的 `<title>` 標籤（50-60 字元）
- [ ] 撰寫吸引人的 meta description（150-160 字元）
- [ ] 加入適當的 meta keywords
- [ ] 設定 canonical URL
- [ ] 加入 viewport meta 標籤
- [ ] 設定正確的語言標籤（lang="zh-TW"）

### Open Graph 標籤
- [ ] 設定 og:title
- [ ] 設定 og:description
- [ ] 設定 og:image（1200x630px）
- [ ] 設定 og:url
- [ ] 設定 og:type
- [ ] 設定 og:site_name
- [ ] 設定 og:locale

### Twitter Card 標籤
- [ ] 設定 twitter:card
- [ ] 設定 twitter:title
- [ ] 設定 twitter:description
- [ ] 設定 twitter:image
- [ ] 設定 twitter:site（如果有）

### 內容優化
- [ ] 使用一個 H1 標籤
- [ ] 使用適當的標題層級（H2-H6）
- [ ] 在標題中包含關鍵字
- [ ] 內容長度至少 300 字
- [ ] 關鍵字自然分布
- [ ] 加入內部連結
- [ ] 加入外部權威連結
- [ ] 內容原創且有價值

### 圖片優化
- [ ] 所有圖片都有 alt 文字
- [ ] 圖片檔案名稱具描述性
- [ ] 圖片已壓縮優化
- [ ] 使用適當的圖片格式（WebP、JPEG、PNG）
- [ ] 大圖片使用 lazy loading
- [ ] 設定圖片尺寸（width 和 height）

### URL 優化
- [ ] URL 簡短且具描述性
- [ ] URL 包含關鍵字
- [ ] 使用連字號分隔單字
- [ ] 使用小寫字母
- [ ] 避免特殊字元和參數

### 結構化資料
- [ ] 加入適當的 Schema.org 標記
- [ ] 使用 JSON-LD 格式
- [ ] 通過 Google Rich Results Test
- [ ] 加入麵包屑標記（如適用）

### 效能優化
- [ ] 頁面載入時間 < 3 秒
- [ ] 圖片已優化
- [ ] CSS 和 JS 已最小化
- [ ] 啟用瀏覽器快取
- [ ] 使用 CDN（如適用）

### 行動裝置
- [ ] 響應式設計
- [ ] 通過 Google Mobile-Friendly Test
- [ ] 觸控元素大小適當（至少 48x48px）
- [ ] 字體大小適當（至少 16px）
- [ ] 避免使用 Flash

### 可訪問性
- [ ] 適當的顏色對比度
- [ ] 鍵盤導航可用
- [ ] ARIA 標籤（如需要）
- [ ] 跳過導航連結
- [ ] 表單標籤正確

## 🔍 技術 SEO 審核清單

### 網站架構
- [ ] 網站有清晰的層級結構
- [ ] 重要頁面距離首頁不超過 3 次點擊
- [ ] 404 錯誤頁面已設定
- [ ] 301 重定向設定正確
- [ ] 沒有損壞的連結
- [ ] 沒有孤立頁面

### HTTPS 和安全性
- [ ] 網站使用 HTTPS
- [ ] SSL 憑證有效
- [ ] HTTP 到 HTTPS 的 301 重定向
- [ ] 混合內容問題已解決
- [ ] 安全標頭已設定

### XML Sitemap
- [ ] XML sitemap 已建立
- [ ] Sitemap 包含所有重要頁面
- [ ] Sitemap 已提交到 Google Search Console
- [ ] Sitemap 已提交到 Bing Webmaster Tools
- [ ] Sitemap 在 robots.txt 中聲明
- [ ] Sitemap 定期更新

### Robots.txt
- [ ] robots.txt 文件存在
- [ ] 允許爬取重要內容
- [ ] 禁止爬取私密和管理頁面
- [ ] Sitemap 位置已聲明
- [ ] 使用 Google Robots Testing Tool 測試

### 索引狀態
- [ ] 重要頁面已被索引
- [ ] 不重要頁面使用 noindex
- [ ] 沒有重複內容問題
- [ ] Canonical 標籤設定正確
- [ ] 分頁設定正確（如適用）

### 網站速度
- [ ] PageSpeed Insights 分數 > 90
- [ ] First Contentful Paint < 1.8s
- [ ] Largest Contentful Paint < 2.5s
- [ ] Cumulative Layout Shift < 0.1
- [ ] Time to Interactive < 3.8s
- [ ] Total Blocking Time < 200ms

### Core Web Vitals
- [ ] LCP（最大內容繪製）< 2.5 秒
- [ ] FID（首次輸入延遲）< 100 毫秒
- [ ] CLS（累積版面配置轉移）< 0.1

### 行動裝置優化
- [ ] 響應式設計或獨立行動版
- [ ] 通過 Mobile-Friendly Test
- [ ] 行動版載入速度快
- [ ] 避免侵入式插頁式廣告
- [ ] 觸控元素間距適當

### 國際化（如適用）
- [ ] hreflang 標籤設定正確
- [ ] 每個語言版本有獨立 URL
- [ ] 內容完全翻譯（不是機器翻譯）
- [ ] 語言切換器易於使用

## ✅ 快速檢查清單（新頁面）

在發布新頁面前，快速檢查以下項目：

1. [ ] Title 標籤（唯一、包含關鍵字、50-60 字元）
2. [ ] Meta description（吸引人、150-160 字元）
3. [ ] H1 標籤（一個、包含關鍵字）
4. [ ] URL（簡短、描述性、包含關鍵字）
5. [ ] 圖片 alt 文字（所有圖片）
6. [ ] 內部連結（至少 2-3 個）
7. [ ] 結構化資料（如適用）
8. [ ] Open Graph 標籤
9. [ ] 行動裝置友善
10. [ ] 頁面速度 < 3 秒
